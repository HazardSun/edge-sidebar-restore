# Edge 侧边栏恢复

恢复 Edge 浏览器被取消的侧边栏功能 — 基于 Edge 原生 Side Panel API，浏览器级面板**零遮挡**、分屏浏览、多功能工具箱。

## 功能

### 侧边栏（原生面板）
- **零遮挡**：使用 Edge 官方 Side Panel API，浏览器真正缩小视口，任何网页（含 fixed 导航、100vw 布局）都不会被遮挡
- **三种打开方式**：点击工具栏扩展图标 / 快捷键 `Ctrl+Shift+Y`（可在 `edge://extensions/shortcuts` 修改）/ 右键扩展图标选择"在侧边栏中打开"
- **页面快捷入口**：网页右边缘有 4px 细条，鼠标悬停滑出窄栏（展开按钮 + 已收藏网站图标），点击即打开原生面板并直达对应网站
- **原生体验**：面板宽度拖拽边缘自由调整，关闭按钮由浏览器提供，跨标签页共享同一面板
- **分屏浏览**：侧边栏内的链接在内嵌浏览器视图中打开，无需离开当前页面

### 工具箱
| 工具 | 说明 |
|---|---|
| 搜索 | Bing / Google 引擎切换，支持常用网站快捷访问（可编辑） |
| 计算器 | 基础运算（+−×÷），键盘快捷键支持 |
| 单位换算 | 长度、重量、温度、面积、体积、速度 6 类单位 |
| 实时汇率 | 28 种货币汇率转换，交叉汇率本地计算即切即显（双数据源自动回退） |
| 快速链接 | 可增删的常用网址收藏，与搜索栏、新标签页共用 |
| 笔记 | 自动保存的便签 |
| 时钟 | 实时时间显示 + 时段问候 + 每日备忘 |

### 新标签页
- 覆盖 Edge 默认新标签页，集成搜索框 + 可编辑快捷链接 + 内嵌侧边栏
- 内嵌侧边栏可通过分隔条上的 › 按钮折叠、右侧 ◀ 按钮展开

## 安装

1. 打开 Edge 浏览器，进入 `edge://extensions/`
2. 开启 **开发者模式**（左上角开关）
3. 点击 **加载解压缩的扩展**，选择本文件夹
4. 或拖拽 `.zip` 压缩包到扩展管理页面

### 使用
- 点击 Edge 工具栏的扩展图标（或按 `Ctrl+Shift+Y`）打开/关闭侧边栏
- 拖拽面板左边缘调整宽度

## 技术说明

- Manifest V3，不上架商店，纯本地扩展
- 侧边栏使用 Edge 原生 `chrome.sidePanel` API（v1.0.0 起；旧版为 Content Script 注入方案，无法避免遮挡网页）
- 无需 `host_permissions`，不读取任何网页内容
- 新标签页通过 `chrome_url_overrides` 覆盖
- 快捷链接、笔记、汇率缓存等数据存储于 `chrome.storage.local`
- 深色/浅色模式跟随系统

## 兼容性

- Microsoft Edge 114+（Side Panel API）

## 隐私

- **无数据收集**，所有数据（笔记、链接、设置）仅存储在本地
- 汇率数据源：fawazahmed0/currency-api（jsDelivr CDN）为主、open.er-api.com 为备份，仅使用时请求并本地缓存 30 分钟
