# Edge 侧边栏恢复

恢复 Edge 浏览器被取消的侧边栏功能 — 常驻显示、分屏浏览、多功能工具箱。

## 功能

### 侧边栏
- **常驻显示**：在任意网页右侧嵌入侧边栏，支持折叠/展开
- **分屏浏览**：侧边栏内的链接在内嵌浏览器视图中打开，无需离开当前页面
- **拖拽调宽**：侧边栏左边缘可拖拽调整宽度（160px–600px）

### 工具箱
| 工具 | 说明 |
|---|---|
| 搜索 | Bing / Google 引擎切换，支持常用网站快捷访问（可编辑） |
| 计算器 | 基础运算（+−×÷），键盘快捷键支持 |
| 单位换算 | 长度、重量、温度、面积、体积、速度 6 类单位 |
| 实时汇率 | 28 种货币实时汇率转换（数据来源：open.er-api.com） |
| 快速链接 | 可增删的常用网址收藏，与搜索栏共用 |
| 笔记 | 自动保存的便签 |
| 时钟 | 实时时间显示 + 时段问候 + 每日备忘 |

### 新标签页
- 覆盖 Edge 默认新标签页，集成搜索框 + 可编辑快捷链接 + 侧边栏
- 支持侧边栏折叠/展开

## 安装

1. 打开 Edge 浏览器，进入 `edge://extensions/`
2. 开启 **开发者模式**（左上角开关）
3. 点击 **加载解压缩的扩展**，选择本文件夹
4. 或拖拽 `.zip` 压缩包到扩展管理页面

### 首次使用
- 安装后侧边栏默认为**折叠状态**，页面右侧显示 ◀ 浮动按钮
- 点击 ◀ 按钮展开侧边栏
- 点击侧边栏标题栏的 ◀ 折叠 按钮或 ✕ 按钮收起
- 点击 Edge 工具栏的扩展图标可切换侧边栏显示


## 技术说明

- Manifest V3，不上架商店，纯本地扩展
- 侧边栏通过 Content Script 注入 `<iframe>` 实现
- 新标签页通过 `chrome_url_overrides` 覆盖
- 折叠/展开/快捷链接等数据存储于 `chrome.storage.local`，跨页面保持
- 深色/浅色模式跟随系统

## 兼容性

- Microsoft Edge 119+
- Google Chrome 119+（理论上兼容，未全面测试）

## 隐私

- **无数据收集**，所有数据（笔记、链接、设置）仅存储在本地
- 汇率数据从 open.er-api.com 获取，仅使用时请求
